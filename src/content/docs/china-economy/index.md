# China Economy

China’s economy is often explained abroad through headlines: property crisis, exports, deflation, overcapacity, stimulus, youth unemployment, industrial policy. Each headline is partly true. None is enough.

This topic tracks the harder question underneath those headlines: how China’s old growth model is being replaced, which parts of the economy are still absorbing the property bust, and which new forces are changing the demand for capital, energy, chips, labor, and strategic resources.
