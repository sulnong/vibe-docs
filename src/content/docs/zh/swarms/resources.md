# 资源与更新追踪

## 去哪里看

| 需要 | 资源 |
| --- | --- |
| 安装和第一次运行 | [Docs](https://docs.swarms.world/) / [PyPI](https://pypi.org/project/swarms/) |
| 源码和问题 | [Repository](https://github.com/kyegomez/swarms) |
| 快速索引 | [llms.txt](https://docs.swarms.world/llms.txt) |

本地笔记只记录项目决策：版本 pin、provider、启用 workflow、工具契约、输出 schema、部署和回滚。

## 按问题类型使用资源

官方文档适合确认支持行为，包信息适合确认要求，源码适合确认实现细节，issues 适合看症状和边缘情况。

| 需要 | 先看 |
| --- | --- |
| 安装或版本要求 | [PyPI package](https://pypi.org/project/swarms/) |
| 支持用法 | [Official docs](https://docs.swarms.world/) |
| 实现细节 | Repository source。 |
| 近期边缘问题 | Repository issues 和 changelog。 |

## 本地笔记保持短

项目笔记只记录决策：version pin、provider、启用 workflow、工具契约、输出 schema、部署路径和回滚。不要镜像整套官方文档。
